# Boogu-Image 研究摘要

Boogu-Image-0.1 是 `boogu-project` 发布的开源图像生成与编辑模型族，采用 Apache-2.0 许可证。它提供 Base、Turbo、Edit 和 fp8 版本，重点能力是文生图、图生图编辑、中英文文字渲染、海报/产品图生成以及 prompt rewriting。

```text

```

## 适合做

- 本地研究和模型能力评测。
- 电商商品图、海报、封面、营销图的原型。
- ComfyUI 工作流。
- 内部运营内容生产后台。
- 作为自建图像生成服务的模型候选。

## 不适合直接做

- 不加审核的公开生成平台。
- 声称官方商业 API。
- 低成本高并发转售服务。
- 对主体身份和细节一致性要求极高的自动修图。
- 需要稳定长文本排版的最终设计稿。

## 试跑建议

1. 先跑 Turbo 1K，看机器能否稳定出图。
2. 再跑 Base 1K/2K，评估中文文字和海报效果。
3. 用 Edit 做删除对象、改色、换背景测试。
4. 对 fp8、offload、cache、Torch Compile 分别做对照，不要一开始全开。
5. 建立自己的商品图/海报/OCR/合规评测表。

## 产品化建议

把 Boogu 包成独立推理服务，前面加业务 API、任务队列、参数 preset、内容审核、人工复核和对象存储。不要让业务系统直接拼命令行调用 `inference.py`，也不要允许用户任意传入底层 CLI 参数。
